# Panda Arm

A simple project for controlling and simulating a Panda robotic arm.

## Features

- Control Panda arm joints
- Simulate movements
- Modular and extensible codebase

## Getting Started

1. Clone the repository:
  ```bash
  git clone https://github.com/yourusername/panda_arm.git
  cd panda_arm
  ```
2. Follow setup instructions in the `docs/` folder.
